# balance-report.md — P-002 전투/밸런스 검증 결과

server.js owner 측정. 헤드리스 자동플레이 하니스 `node server.js --balance`
(4직업 × 8시드 × 2모드: 무보강=난이도 바닥 / 증강=실제 플레이). 봇=직접조준·카이팅·windup 회피.

## 핵심 버그 수정
- **적→플레이어 무피해 버그**: 플레이어 객체에 충돌 반지름 `p.r` 누락 → `e.r + p.r` = NaN →
  적 접촉/투사체 판정이 항상 false였음. `addPlayer`에서 `r = CHAMPIONS[champion].r` 부여로 해결.
  소켓 재현: 가만히 선 플레이어 HP 150→109(12s), 대응 안 하면 사망(헤드리스 23.5s).

## 적 공격 모션(텔레그래프)
- 적에 접근→**windup(예고)**→**strike(타격)**→**recover(복귀)** FSM. 데미지/투사체는 strike 시점에만 발생.
- snapshot 노출: `enemy.state`(move/windup/strike/recover/charge), `enemy.attackAnim{type,startedAt,windup,strike,duration,aimAngle}`,
  이벤트 `enemy_attack{...}`. aimAngle은 startedAt에 고정(추적 금지) → 텔레그래프 방향=타격 방향 일치.
  (모션 애니메이터·프론트·VFX 합의 규격)
- 적별 windup/duration(ms): 고블린280/520, 박쥐240/480, 슬라임360/600, 오크480/760, 실드오크520/820,
  해골500/700, 슬링거460/660, 다크메이지560/780, 힐러임프600/820, 거인460/720, 보스700~ / 하한 240.

## 합격선 면제 1건(게임 기획자 승인)
- **TTK 하한 0.2s 면제(최약체 잡몹 한정)**: 전사·암살의 고블린/박쥐 1타 즉사(0.03s)는
  뱀파이어서바이버류 표준 손맛이고, 잡몹 HP는 이미 상향·슬라임은 0.5~0.8s 정상, 공격 쿨다운 0.5s가
  연사 폭주를 막으므로 면제. (슬라임 TTK 전사0.57/마법0.63/궁수0.80s 정상)
- 단, **1타 즉사여도 공격 모션은 끝까지 재생**: 검증됨 — 암살자 dash 모션(animDur 0.30s)이
  적 즉사(strike 프레임 사망) 후에도 9tick(0.30s) 동안 유지(`attackAnim`은 animDur 경과 시에만 해제).
